# UltraCompany Static Website

Website statis promosi perusahaan dengan beberapa halaman:

- `index.html` sebagai beranda utama.
- `about.html` untuk profil perusahaan.
- `services.html` untuk daftar layanan.
- `contact.html` untuk kontak.

## Menjalankan

1. Buka `index.html` di browser, atau
2. Jalankan server statis dari folder root proyek.

## Catatan deploy

- Semua aset menggunakan path relatif.
- Tidak ada dependensi JavaScript.
- Cocok untuk hosting statis seperti GitHub Pages, Netlify, atau shared hosting.
